三种模式 

类别	操作说明	命令（均在「普通模式」下输入）	备注/示例	
光标移动	上下左右	`h` `j` `k` `l`	左下上右	
	行首/行尾	`^` / `0` 与 `$`	`0` 绝对行首，`^` 非空行首	
	文件首尾	`gg` / `G`	首行 / 末行	
	指定行	`:42` 或 `42G`	跳到第 42 行	
	单词跳转	`w` `W` / `b` `B`	下一/上一单词（大写含标点）	
	段落/括号	`{` `}` / `%`	段落首尾；`%` 匹配括号	
	屏幕位置	`H` `M` `L`	屏幕顶/中/底行	
查找	向下查找	`/pattern` + `Enter`	`n` 下一个，`N` 反向	
	向上查找	`?pattern` + `Enter`	同上	
	忽略大小写	`:set ic`	`/foo` 匹配 FOO	
	高亮开关	`:set hls` / `:noh`	高亮/取消高亮	
修改	删除字符	`x` / `X`	删光标处/前	
	删除单词	`dw` / `dW`	含/不含标点	
	删除整行	`dd`	同时存入默认寄存器	
	删除到行尾	`D` 或 `d$`		
	撤销/重做	`u` / `Ctrl-r`		
	替换单个字符	`r` + 新字符	无需进插入模式	
	进入插入模式	`i` `I` `a` `A` `o` `O`	前/后/下新/上新	
复制/粘贴	复制行	`yy` 或 `Y`	整行	
	复制多行	`3yy`	复制 3 行	
	复制区域	`v` 移动 `y`	可视模式选完按 `y`	
	粘贴	`p` / `P`	后/前	
	查看寄存器	`:reg`	`"ayy` 复制到 a 寄存器，`"ap` 粘贴	
替换	当前行替换	`:s/old/new/`	仅第一个	
	当前行全替换	`:s/old/new/g`		
	全文替换	`:%s/old/new/g`		
	交互替换	`:%s/old/new/gc`	每次询问	
	区域替换	`:'<,'>s/old/new/g`	可视模式选区后	
保存/退出	保存	`:w`		
	另存为	`:w newfile`		
	强制保存	`:w !sudo tee %`	只读文件提权	
	退出	`:q`		
	强制退出	`:q!`	放弃修改	
	保存并退出	`:wq` 或 `ZZ`		
	全部保存退出	`:xa` 或 `:wqa`	多文件时	
其他常用	显示行号	`:set nu` / `:set nonu`		
	分屏	`:sp` / `:vsp`	横/纵分	
	切换窗口	`Ctrl-w w` / `h/j/k/l`		
	执行 shell	`:!command`	`:!ls`	
	帮助	`:help`	`:help :s`	


$0为脚本名字
expr关系运算符只支持简单的< = !=不支持>=
let没有空格， expr有 
expr的必须转义 \*  \% 需要$取值   let不需要取值
?匹配一个  * 0个或多个
